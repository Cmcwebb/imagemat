This directory contains server side development code relating to end user registration You probably either came here in error or wish to
  1. Register to use ImageMAT
  2. Logon to ImageMAT
  3. Change your profile
  4. Check your time settings
  5. Reset a forgotten password
  6. Log out
  7. Verify an email sent you
  8. Cancel the verification process
  9. Ask who you are currently logged in as
  10. Delete yourself
  11. Change your default permissions
